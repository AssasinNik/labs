### Почему IPv6 является подходящим протоколом для IoT?

**Ответ:** Он обеспечивает значительно увеличенное количество адресов (по сравнению с IPv4)

---

### Какое поле в заголовке IP призвано предотвратить бесконечное блуждание пакета между сетями?

**Ответ:** TTL (Time-To-Live)

---

### В чем различие между протоколами TCP и UDP в плане надежности доставки?

**Ответ:** TCP обеспечивает проверку на наличие ошибок доставки, UDP – нет

---

### Какой протокол предоставляет услуги безопасности CoAP?

**Ответ:** DTLS (Datagram Transport Layer Security)

---

### Протокол ARP предназначен для:

**Ответ:** Определения MAC-адреса по IP-адресу другого компьютера

---

### На каком уровне модели OSI осуществляется фрагментация пакетов?

**Ответ:** Транспортный уровень.

---

### Соотнесите перечисленные протоколы и уровень, к которому они относятся: UDP, TCP, DTLS

**Ответ:** Transport (Транспортный уровень)

### Модель OSI имеет 7 уровней, расставьте их в правильном порядке, начиная с нижнего.

**Ответ:** 2463571.

1. Physical (Физический) — 2
2. Data Link (Канальный) — 4
3. Network (Сетевой) — 6
4. Transport (Транспортный) — 3
5. Session (Сеансовый) — 5
6. Presentation (Уровень представления) — 7
7. Application (Прикладной) — 1
